# PC 로컬 교사용 위젯 대시보드 (Class Widget Space)

본 애플리케이션은 **Windows PC 바탕화면 위젯 스타일**로 작동하는 교사용 로컬 단말 대시보드 프로그램입니다. 교과 수업 진행, 학생 관리(출결/칭찬/상담), 학급 일정 조율 및 공문용 시간표 관리를 한곳에서 처리하며, 학교 현장 데이터의 보완을 위해 **100% 오프라인 브라우저/로컬 저장소**만 사용합니다.

---

## 📸 핵심 기능 구성

총 10가지 이상의 Apple 위젯 디자인 카드형 위젯 패널이 탑재되어 있습니다.

1. **시계 & 날짜 위젯**
   - 아날로그 원형 시계페이스 및 정밀 디지털 24H 시계 지원 (1초 단위 째깍 작동).
2. **생활 날씨 위젯**
   - 설정 지역(서울, 부산, 인천 등) 기준 생활 대기 정보 및 예시 날씨 시각화.
3. **수업 보조 도구 모음**
   - **학생 뽑기**: 학급 명렬 데이터와 100% 동화 연관된 1명 및 N인 무작위 추첨기.
   - **시간 타이머**: presets 버튼(+1m, +5m, Stop)이 탑재된 실제 1초(1000ms) 정밀 차트 타이머.
   - **모둠 점수판**: 가산식 터치 스코어 카드 완비.
   - **룰렛 돌림판**: 로터리 캔버스로 구성되어 명목 배치 후 회전하는 고유 캡슐 게임.
   - **QR 코드 생성기**: 학교 기밀용으로 외부 전송 없이 완벽히 로컬 캔버스에 그리는 오프라인 QR 배너.
4. **오늘의 할 일 위젯**
   - 카테고리 태그 분류(행정업무, 수업준비, 생활지도...)와 체크 수집 토글러.
5. **오늘의 학교 급식 위젯**
   - NEIS Open API 연동 전국 소속 학교 식사 매칭.
   - 영양 알레르기 수치(`1.2.5...`)를 친절한 국문 한글 명칭('난류', '우유', '대두')으로 실시간 변환.
   - 지나간 날과 앞으로 올 모든 날들로 일자 이동 지원 및 오프라인 예시 식단 전환 탑재.
6. **학사일정 & 월간 달력 위젯**
   - 달력 클릭 시 간이 메모 등재 지원 및 `.ics` 파일 수집기 탑재.
7. **학생 명렬표 & 관리 위젯**
   - 다중 학급 생성 및 번호순 학적 정보 기재.
   - 12종 출결 양식(출석, 질병지각, 미인정결석 등) 및 비공개 가명 상담 일지 관리.
   - 칭찬 스티커 가감, 엑셀 템플릿(Roster) 다운로드 및 CSV 대량 소환/수출 지원.
8. **시험 & 행사 D-Day 위젯**
   - 학기 고사, 주요 연수, 자문회 일정 countdown 표 및 완제 체크 박스.
9. **포스트잇 메모 위젯**
   - 5종 파스텔 애플 캔버스 컬러 스위치 탑재.
10. **학급 및 교사 수기 시간표 위젯**
    - 월~금 1~7교시 단위 그리드 연산식 내역 더블클릭 기재 및 엑셀 Import/Export.

## 🛠️ 개발 및 로컬 실행 로드맵

Vite 와 React 환경을 기본으로 구동하며, 추후 Electron 프레임 탑재가 가능한 데스크톱 포맷 규격을 충족합니다.

### 1) 의존 라이브러리 검출 설치
```bash
npm install
```

### 2) 개발 전용 서버 로컬 구동 (Port: 3000)
```bash
npm run dev
```

### 3) 로컬 프로덕션 Static 빌드
```bash
npm run build
```

## 🛡️ 개인정보 수집 및 로컬 프라이버시 준수 선언
- 모든 데이터(학생명, 상담 내용, 출석 여부, 교직 시간 설정)는 외부 클라우드로 단 1Byte도 유출되지 않으며 오직 사용자의 브라우저 또는 PC 단말 **LocalStorage 환경에서만 분리 기장**됩니다.
- NEIS OpenAPI 통신은 브라우저에서 CORS 가 공식 해결된 허브와 동시  fetch 되어 기기에 영구 안전하게 동화됩니다.
